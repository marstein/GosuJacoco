JaCoCo

JaCoCo - Java Code Coverage Library

JaCoCo is a free Java code coverage library implemented by Marc R. Hoffmann and Brock Janiczak. Check http://www.eclemma.org/devdoc/jacoco.html for updates, documentation and support.

This is the distribution of version @VERSION@.

Implementation Status

This version is a plain prototype to evaluate implementation strategies and verify the basic ideas. Please refer to the documentation for details. This is not a stable library or usable tool yet!
