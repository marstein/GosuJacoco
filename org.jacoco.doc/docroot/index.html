JaCoCo

JaCoCo - Java Code Coverage Library

JaCoCo is a free Java code coverage library implemented by Marc R. Hoffmann and Brock Janiczak. Check @HOMEURL@ for updates and feedback.

This is the distribution of version @VERSION@.

Implementation Status

This version is a plain prototype to evaluate implementation strategies and verify the basic ideas. Please refer to the documentation for details. This is not a stable library or fully usable tool yet!

File Description Dependencies
jacocoagent.jar Java agent for execution data recording None, all dependencies included

org.jacoco.core_@VERSION@.jar JaCoCo core bundle ASM 3.x
org.jacoco.report_@VERSION@.jar JaCoCo report bundle JaCoCo core, ASM 3.x
org.jacoco.ant_@VERSION@.jar JaCoCo Ant bundle JaCoCo core, JaCoCo report, ASM 3.x, Ant 1.7.x
