JaCoCo

JaCoCo - Java Code Coverage Library

JaCoCo is a free Java code coverage library implemented by Marc R. Hoffmann and Brock Janiczak distributed under the Eclipse Public License. Check @HOMEURL@ for updates and feedback.

This is the distribution of version @VERSION@.

Implementation Status

This first official release of JaCoCo is a fully functional prototype to verify the basic concepts and implementation strategies. It already produces neat coverage reports and has been successfully tested with huge Eclipse RCP applications and web projects.

Due to the prototype status future versions will come with additional or modified features and may show different behavior in certain situations. The Java API is not frozen yet.
